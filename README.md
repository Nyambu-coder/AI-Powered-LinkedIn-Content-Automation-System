# 🚀 AI-Powered LinkedIn Content Automation System

## 💼 Overview
I designed and built a fully automated LinkedIn content engine that eliminates manual posting, ensures consistency, and transforms content creation into a scalable system.

This workflow runs daily without manual input and replicates a real-world content marketing pipeline used by creators, founders, and agencies.

---

## 🎯 The Problem
Maintaining a consistent LinkedIn presence is difficult:

- Time-consuming content creation (1–2 hours per post)
- Lack of consistency
- Idea-to-post friction
- Manual publishing process

---

## 💡 The Solution
A fully automated system that:

- Pulls content ideas from Notion
- Uses AI to generate high-quality posts
- Ensures media is attached before publishing
- Uploads images via LinkedIn API
- Publishes posts automatically
- Prevents duplicate content

---

## ⚙️ How It Works

### 🔁 Workflow Pipeline

Trigger → Notion → AI Generation → Validation → Media Upload → LinkedIn Post → Status Update

---

## 🧠 Tech Stack

- n8n (workflow automation)
- OpenAI (content generation)
- LinkedIn API (UGC Posts + Media Upload)
- Notion API (content management)
